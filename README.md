# HABEX Real Estate

Sitio web de **HABEX Real Estate** — desarrollos de inversión inmobiliaria (lotes,
departamentos, casas y macro terrenos) en Yucatán y México.

Sitio estático (HTML/CSS/JS, sin build) publicado con **GitHub Pages**.

## Verlo localmente

```bash
python3 -m http.server 8000   # luego abre http://localhost:8000
```

## Publicación

Se sirve automáticamente desde la rama `main` vía GitHub Pages en cada push.
